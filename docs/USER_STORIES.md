# User Stories

## Customer User Stories

### Scan QR Code

As a customer,
I want to scan a QR code,
so that I can access the restaurant menu without waiting for a waiter.

---

### Browse Menu

As a customer,
I want to view food categories and items,
so that I can quickly find what I want to order.

---

### Add Items to Cart

As a customer,
I want to add food items to my cart,
so that I can place multiple orders at once.

---

### Modify Cart

As a customer,
I want to increase or decrease item quantities,
so that I can change my order before checkout.

---

### Place Order

As a customer,
I want to place my order digitally,
so that the kitchen receives it immediately.

---

### Track Order Status

As a customer,
I want to see whether my order is preparing or ready,
so that I know when my food will arrive.

---

## Kitchen Staff User Stories

### View Incoming Orders

As a kitchen staff member,
I want to see all active orders,
so that I can prepare food efficiently.

---

### Update Order Status

As a kitchen staff member,
I want to change order status,
so that customers receive real-time updates.

---

### View Order History

As a kitchen staff member,
I want to access completed orders,
so that I can review previous activities.

---

## Admin User Stories

### Login Securely

As a restaurant owner,
I want secure authentication,
so that only authorized users can access the dashboard.

---

### Manage Menu

As a restaurant owner,
I want to add, edit, and remove food items,
so that I can keep my menu updated.

---

### Update Prices

As a restaurant owner,
I want to change food prices,
so that I can adjust my business offerings.

---

### Generate QR Codes

As a restaurant owner,
I want to generate QR codes for tables,
so that customers can access the menu instantly.

---

### View Analytics

As a restaurant owner,
I want to see sales and order analytics,
so that I can make better business decisions.
